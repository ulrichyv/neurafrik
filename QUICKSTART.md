# ⚡ Démarrage Rapide

## 1️⃣ Installation (30 secondes)

```bash
# Allez dans le dossier du projet
cd "c:\Users\ulrich\Downloads\Nouveau dossier (15)\stitch\neurafrik_site"

# Installez les dépendances
npm install
```

## 2️⃣ Lancer le serveur (1 seconde)

```bash
npm start
```

Vous verrez:
```
🚀 Neurafrik Server running at http://localhost:3000
```

## 3️⃣ Ouvrir dans le navigateur

**http://localhost:3000**

## 🎉 C'est prêt!

Naviguez entre les pages:
- 🏠 [Home](http://localhost:3000/home.html)
- ℹ️ [About](http://localhost:3000/about.html)
- 🔧 [Services](http://localhost:3000/service.html)
- 💡 [Solutions](http://localhost:3000/solution.html)
- 📧 [Contact](http://localhost:3000/contact.html)

## 📝 Tester le formulaire

1. Allez à http://localhost:3000/contact.html
2. Remplissez le formulaire de contact
3. Cliquez "Send Request"
4. Vérifiez la réponse de succès

Les données sont stockées sur le serveur et visibles via:
- `GET http://localhost:3000/api/submissions`

---

**Pour arrêter le serveur**: Appuyez sur `Ctrl + C`
